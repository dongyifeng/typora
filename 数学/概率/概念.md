# 条件概率和独立性

$P(A|B)=\frac{P(A\cap B)}{P(B)}$







# 统计



均值 ( mean )

$E(X) = \sum_{i=1}^{n}x_i*p_i$

```python
import numpy as np
import math

data = [1, 2, 3, 4]
np.mean(data)

mean = 1.0 * sum(data) / len(data)
```

方差 ( Variance )

$D(x) = Var(X) =E[X-E(X)]^2=E(X^2)-E[(X)]^2$

```python
print(np.var(data))

# 或者
s = 0
s1 = 0
for item in data:
    s += item
    s1 += item * item

l = len(data) * 1.0
var = s1 / l - math.pow(s / l, 2)
```
