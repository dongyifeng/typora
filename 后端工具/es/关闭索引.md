# 关闭索引

关闭 commodity 索引

```react
POST http://127.0.0.1:9200/commodity/_close
```

结果：

```json
{
  "acknowledged":true
}
```



# 打开索引

```react
POST http://127.0.0.1:9200/commodity/_open
```

结果：

```json
{
  "acknowledged":true,
  "shards_acknowledged":true
}
```

文档被打开成功后，我们就可以愉快的对索引进行读写操作了。
