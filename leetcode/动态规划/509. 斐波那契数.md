# 斐波那契数

> **斐波那契数**，通常用 `F(n)` 表示，形成的序列称为 **斐波那契数列** 。该数列由 `0` 和 `1` 开始，后面的每一项数字都是前面两项数字的和。也就是：

```python
F(0) = 0，F(1) = 1
F(n) = F(n - 1) + F(n - 2)，其中 n > 1
```

给你 `n` ，请计算 `F(n)` 。

**示例 1：**

```python
输入：2
输出：1
解释：F(2) = F(1) + F(0) = 1 + 0 = 1
```



**示例 2：**

```python
输入：3
输出：2
解释：F(3) = F(2) + F(1) = 1 + 1 = 2
```



**示例 3：**

```python
输入：4
输出：3
解释：F(4) = F(3) + F(2) = 2 + 1 = 3
```

**代码：**

```python
def fib(n):
    if n < 2: return n
    num1 = 0
    num2 = 1
    res = 0
    for i in range(1, n):
        res = num1 + num2
        num1 = num2
        num2 = res
    return res
```



```python
def fib2(n):
    if n < 2: return n
    return fib2(n - 1) + fib2(n - 2)
```
