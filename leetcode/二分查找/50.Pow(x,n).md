> 实现 [pow(*x*, *n*)](https://www.cplusplus.com/reference/valarray/pow/) ，即计算 x 的 n 次幂函数（即，xn）。

**示例 1：**

```python
输入：x = 2.00000, n = 10
输出：1024.00000
```

**示例 2：**

```python
输入：x = 2.10000, n = 3
输出：9.26100
```

**示例 3：**

```python
输入：x = 2.00000, n = -2
输出：0.25000
解释：2-2 = 1/22 = 1/4 = 0.25
```



**代码：**

```python
'''
折半

原理：x ^(n+m) = x^n * x^m
将指乘积转化为加法
'''
def myPow(x, n):
    if n == 0: return 1.0
    memo = {1: x}
    res = inter_pow(x, abs(n), memo)
    return res if n > 0 else 1.0 / res


def inter_pow(x, n, memo):
    if n in memo: return memo[n]
    mid = n >> 1

    l = inter_pow(x, mid, memo)
    memo[mid] = l
    r = inter_pow(x, n - mid, memo)
    memo[n - mid] = r
    return l * r
```
