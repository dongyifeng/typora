> 给你一个非负数整数n，判断n是不是一些数（这些数不允许重复使用，且为正数）的阶乘之和，如9=1！+2!+3!，如果是，则输出Yes，否则输出No；



**代码：**

```python
def factorial_sum(n):
    res = 0
    factorial = 1
    for i in range(1, n):
        if res == n: return True
        if res > n: return False
        factorial *= i
        res += factorial


print(factorial_sum(9))
print(factorial_sum(10))
```
