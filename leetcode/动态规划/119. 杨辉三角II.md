# 杨辉三角II

> 给定一个非负索引 `rowIndex`，返回「杨辉三角」的第 `rowIndex` 行。
>
> 在「杨辉三角」中，每个数是它左上方和右上方的数的和。

![](images/118_1.gif)

**示例 1:**

```python
输入: rowIndex = 3
输出: [1,3,3,1]
```



**示例 2:**

```python
输入: rowIndex = 0
输出: [1]
```



**示例 3:**

```python
输入: rowIndex = 1
输出: [1,1]
```



**代码：**

```python
def generate(num_rows):
    if num_rows <= 0: return [1]
    lash_row = [1]
    for i in range(1, num_rows+1):
        new_row = [1]
        for j in range(1, i):
            new_row.append(lash_row[j - 1] + lash_row[j])
        new_row.append(1)
        lash_row = new_row
    return lash_row
```
