> 实现 [pow(*x*, *n*)](https://www.cplusplus.com/reference/valarray/pow/) ，即计算 x 的 n 次幂函数（即，xn）。

**示例 1：**

```python
输入：x = 2.00000, n = 10
输出：1024.00000
```

**示例 2：**

```python
输入：x = 2.10000, n = 3
输出：9.26100
```

**示例 3：**

```python
输入：x = 2.00000, n = -2
输出：0.25000
解释：2-2 = 1/22 = 1/4 = 0.25
```



**代码：**

```python
'''
折半

原理：x ^(n+m) = x^n * x^m
将指乘积转化为加法
'''
def myPow3(x, n):
    res = x
    i = n
    while i > 0:
        # 奇数：减一变为偶数
        if i % 2 != 0:
            i -= 1
            res *= x
        # 偶数：减半
        else:
            i >>= 1
            res = res * res
	
  	# 处理 n 为负数
    return res if n > 0 else 1.0 / res
```
